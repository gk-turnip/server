This is the Gourdian Knot game currently under development. Please try it out, see how it works, play around. Check out the main site and forum for any questions or if you are interested in helping out or interested in seeing our work in progress.

FOR NEW USERS: If you are a new user you can easily create an account, just click on the Register button.

Working browser list:
  • Firefox version 21
